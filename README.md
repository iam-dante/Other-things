제가 꼭 성공 후 정리해서 글을 써보겠읍니다.
  
- 맞춤법 되와 돼가 간간히 헷갈려서 글을 찾아 봤습니다. 세가지 방법(?)이 있더군요.
  - '되'는 '하'로 '돼'는 '해'로 바꿔서 더 자연스러운게 맞다는군요.
   ```
   밥 먹어야 돼요. > 밥 먹어야 해요. (O)
   밥 먹어야 되요. > 밥 먹어야 하요. (X)
   ```
  - '돼'를 '되어'로 풀어 사용해봅니다.
  ```
  박사가 돼고 싶다. > 박사가 되어고 싶다. (X)
  나는 아빠가 됐어. > 나는 아빠가 되었어. (O)
  ```
- 문장 마지막에는 무조건 '돼'
  - 이건 저도 알고 있었어요..  
  ```안되.(X) / 안돼.(O)```
